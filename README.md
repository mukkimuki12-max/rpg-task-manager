# rpg-task-manager
タスク管理ツールに育成要素を取り込んだもの。claude codeの作成物。pythonで書いてもらいました。

## 画面
### タスク管理画面
<img width="2553" height="498" alt="image" src="https://github.com/user-attachments/assets/ca7cff87-893c-410a-94ea-4ad2f747707b" />
タスクを作成したり、ステータスをドラッグアンドドロップできるカンバンボード。　　
タスクは完了するとポイントをもらえ、マイページでアバターの装備を買うことができる。

### マイページ
<img width="2477" height="899" alt="image" src="https://github.com/user-attachments/assets/d52df477-a79c-44cd-9e6e-7b5ed3bed791" />
アバターの装備を更新したり、装備を買ったりできる。
装備は着替えるとちゃんと色味が変わる。

## RPGタスクマネージャー 起動コマンド
ターミナル起動
C:\ app.py

起動後、ブラウザで以下にアクセス:
http://localhost:5000

## 停止方法
起動したターミナルで Ctrl+C を押す
